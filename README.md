# D3 JS and React

React Redux Setup for displaying a line chart using D3


# Commands

### Installation

Install:
```bash
npm install
```

### Build & Server

Create Development build & run server on local in development mode:
```bash
npm run start
```

Create production build
```bash
npm run build
```

### Testing
Run unit tests once:
```bash
npm test
```

### Linting

Run js lint:
```bash
npm lint:js
```

Run css lint:
```bash
npm lint:css
```
